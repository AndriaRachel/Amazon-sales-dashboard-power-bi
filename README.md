# Amazon Sales Dashboard 2025

## Project Overview

This project is an interactive Power BI dashboard built using an Amazon Sales 2025 dataset. The goal of the project is to analyze sales performance, identify top-performing products and categories, and visualize key business metrics through an easy-to-understand dashboard.

As an aspiring Data Analyst, I created this project to practice data cleaning, transformation, visualization, and dashboard design while working with a real-world sales dataset.

## Dataset

The dataset used in this project was obtained from Kaggle:

Dataset: Amazon Sales 2025

Source: https://www.kaggle.com/datasets/zahidmughal2343/amazon-sales-2025

The dataset contains sales-related information that can be used to explore trends, product performance, revenue distribution, and other business insights.
